## :checkered_flag: Starting

```bash
# 프로젝트를 복제하세요
$ git clone https://github.com/osh6006/football5.git

# 폴더 이동
$ cd football5

# dependencies 설치
$ npm i

# 프로젝트 실행
$ npm run dev

# 로컬 서버 <http://localhost:5173>으로 시작합니다!
```
